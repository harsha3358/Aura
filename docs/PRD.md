# Product Requirements Document (PRD)

## Problem & Vision
AURA builds an increasingly accurate cognitive model of the user — not a chatbot. North star metric: Hours Saved Per User Per Week.

## Primary Founder Metric
Did AURA save the founder at least 30 minutes today?
YES / NO
Track daily. Before worrying about thousands of users, prove value to yourself.

## MVP Scope (Layers 0–6 only; no Agents)

| Feature | Priority | Success Criteria |
|---------|----------|-------------------|
| Auth + Onboarding | P0 | User verified, profile + initial contexts/projects captured |
| Conversation Capture | P0 | Every message routed through Extraction Engine |
| Knowledge Contract | P0 | Pollution rate < 5%; sub-threshold → observations only |
| Cognitive Memory | P0 | Facts, decisions, tasks, deadlines persisted with context |
| Executive Brief | P0 | Daily brief generated with priorities, deadlines, blockers, actions |
| Living Benchmark | P1 | 500 labeled examples; precision/recall/F1 tracked |
| Hypothesis + Strategy | P1 | Tentative hypotheses generated; strategies from validated only |
| Context Engine | P0 | Context tagged on every extraction; context shift detected |
| Extraction Feedback | P0 | Track explicit user feedback (correct, incorrect, partial). Every incorrect extraction becomes a benchmark candidate. |
