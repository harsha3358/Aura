# Product Requirements Document (PRD)

## Problem & Vision
AURA builds an increasingly accurate cognitive model of the user — not a chatbot. North star metric: Hours Saved Per User Per Week.

## Target Users
Students, founders, engineers, researchers, product builders, professionals.

## MVP Scope
- Auth + Onboarding (P0)
- Conversation Capture (P0)
- Knowledge Contract (P0)
- Cognitive Memory (P0)
- Executive Brief (P0)
- Living Benchmark (P1)
- Hypothesis + Strategy (P1)
- Context Engine (P0)

## Out of Scope (MVP)
- Planner/Worker/Critic agents
- Multi-user collaboration
- Mobile native apps
- Third-party integrations
